# cyber Security Awareness Chatbot

This is a cybersecurity awareness chatbot designed to educate its users about online safety, phishing attacks, password security, and safe browsing practices

The chatbot's features include (but some features are coming soon):

Voice greeting with customizable welcome message
- 🛡️ ASCII art logo and visual elements
- 💬 Interactive conversational interface
- 🔐 Cybersecurity education on various topics
- 🌈 Colorful console UI with typing effects
- 📊 Session tracking and summary
